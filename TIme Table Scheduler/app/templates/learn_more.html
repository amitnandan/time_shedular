So where do we start? Let's see.

For Creating a time table, we need basic information initially, such as

  • TEACHER - Teacher's Name.
  • SUBJECT - Subject's Name.
  • PRACTICAL/LECTURE - Is it a lecture or a practical.
  • WEEKLY COUNT - Weekly Count or Load given to that subject.
  • DAILY COUNT - Daily Count given to that subject.

Step 1: Inserting Data

We've provided 2 ways for accepting the required information.

    1. Manually Insert all the data.

    OR

    2. Upload all of the data at once using Excel File.

Rules! for creating Excel file. Starting from Column 0 Row 0,

  • 0th Column should have Teachers Name.
  • 1st Column should contain respective Teachers Subject.
  • 2nd Column should contain respective Subjects Weekly Count.
  • 3rd Column should have the required Daily Count of the Subject.
  • 4th Column Must have the location of the Lab/Class-room of the Given Subject.

Step 2: Verifying Data

Before Creating a time table, make sure you've uploaded your data correctly.

If something is incorrect or you would simply like to change a teacher's name or anything else, just click on Login at the top-most button on the right & go to Lectures.

Step 3: Time Table

Now that all data has been uploaded and verified, all you have to do is click on the Generate Time Table Button. And the rest will be taken care of. :)
